7. ВЫВОД
--------

акциях: подняли прайс -> товар не прошёл ценовой порог -> выпал из акции ->
потерял акционную скидку целиком -> эффективная цена +20..47% -> ВБ снял с
продвижения (трафик -21%) И конверсия упала. Два эффекта перемножились = -40%.

Склад (Питер) и спрос — не причина: падение равномерное по стране, сток стабилен,
Ozon/Маркет не просели.
