## 1. ЧТО ЭТО
Сбор данных с маркетплейсов + МойСклад → витрина маржи по SKU → веб-дашборд с
советами. Два аккаунта WB: «Цифровой» (wb_acc1) и «ДИСКВЭР» (wb_acc2). Postgres в
docker (mp-postgres:16, порт 5433). Ежедневный сбор run_daily.py по cron 2×/день.
