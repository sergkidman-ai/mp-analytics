## Вход в сессию (шпаргалка)
```
ssh <сервер>
tmux ls                 # список сессий; каноничные имена: fin mkt rev gab inv china
tmux attach -t <имя>    # напр. tmux attach -t mkt
# внутри tmux:  Ctrl+b d — отсоединиться (сессия живёт);  tmux switch-client -t <имя> — перейти в другую
```

## Процедура переезда сессии
Переезд = чистый рестарт Claude в той же tmux-сессии (напр. при переполнении контекста). **Tmux-сессии
при переезде НЕ пересоздаются** — новый `claude` стартует в уже существующем окне того же потока.
1. **Дежурному (перед выходом):** обнови свою секцию потока в этом `HANDOFF.md` (состояние / сделанное /
   активные ветки / открытые вопросы / следующий шаг) и **закоммить всё незакоммиченное** в своей ветке.
2. `/exit` — завершить текущую сессию Claude.
3. `claude` — запустить заново в ТОЙ ЖЕ tmux-сессии (окно потока не трогаем).
4. Стартовое сообщение новой сессии:
   > «Ты сессия потока **X**, прочитай `CLAUDE.md` + свою секцию `HANDOFF` + `BRIEF`, доложи и продолжай».
