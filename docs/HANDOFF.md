# HANDOFF — состояние проекта по потокам

> **Читать ПЕРВЫМ делом при старте сессии** — свою секцию потока (ниже). Обновлять в конце значимого блока.
> Карта потоков и сессий — `docs/SESSIONS_MAP.md`; правила территорий — `CLAUDE.md`. Обновлено: 2026-07-21.
> Каждая сессия ведёт СВОЮ секцию; чужие секции не трогать (нужна чужая территория → пиши вопрос в свою секцию).
> worktree-сессии при старте подтягивают доки из `main` (`git show main:CLAUDE.md` / `git show main:docs/HANDOFF.md`) — веточная копия бывает stale.

## Вход в сессию (шпаргалка)
```
ssh <сервер>
tmux ls                 # список сессий; каноничные имена: fin mkt rev gab inv china
tmux attach -t <имя>    # напр. tmux attach -t mkt
# внутри tmux:  Ctrl+b d — отсоединиться (сессия живёт);  tmux switch-client -t <имя> — перейти в другую
```
