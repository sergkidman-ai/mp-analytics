Wildberries — сводный отчёт по месяцам

Как читать этот отчёт

Данные 1:1 из Финансового отчёта Wildberries (Баланс) личного кабинета (сверено с ЛК: К перечислению Δ<0,05%, Итого к оплате +537 ₽ / +0,03% — в допуске) + операционные показатели (продажи, возвраты, средний чек) и себестоимость из МойСклад. Оборот = «Продажа — наша цена» (цена после нашей скидки, ДО СПП) — от неё считаются все % и подсветка. «ВБ реализовал» — это уже после СПП (что заплатил покупатель); разница = СПП (скидка ВБ за свой счёт). Строки — водопад: наша цена → минус СПП, возврат, комиссия ВБ → К перечислению → минус логистика, хранение, приёмка, удержания → Итого к оплате → минус COGS → Чистая. Каждое юрлицо — своя таблица, не суммируем. Столбцы — месяцы формирования отчётов (весь недельный отчёт падает в свой месяц формирования — модель данных ВБ). Справа — доля от оборота и тренд. Подсветка — три блока относительно среднего: зелёное = выше среднего (хорошо), янтарное = ниже среднего (обратить внимание), без заливки — норма. Для расходов инверсия (ниже — лучше). Два правых столбца — текущий месяц (оценка) и прогноз на конец месяца, живьём из БД.

выше среднего — хорошо ниже среднего — обратить внимание в норме (около среднего) наша не из отчёта МП (МойСклад) расчёт производная над константами

Все строки воспроизводят Финансовый отчёт ВБ → Баланс. Оборот = «Продажа — наша цена» (retail_price_withdisc_rub, ДО СПП). «ВБ реализовал» = retail_amount (после СПП, что заплатил покупатель); «СПП» = наша цена − ВБ реализовал (скидка ВБ за свой счёт, ≈28–29%). «Возврат» в сырье ВБ лежит с положительным ppvz_for_pay — в «К перечислению» он вычитается (иначе завышение 2×). «Комиссия ВБ» = ВБ реализовал − Возврат − К перечислению (реальная удержка площадки, без СПП; формулой не моделируется — берём фактическую разницу). «Прочие удержания» = штрафы + удержания + баллы лояльности (cashback). «Итого расходы ВБ» = наша цена − Итого к оплате (все удержания площадки: СПП + возврат + комиссия + логистика + хранение + приёмка + прочие). «Итого к оплате» = К перечислению − Логистика − Хранение − Приёмка − Прочие. Себестоимость — FIFO из МойСклад по assembly_id (сторно COGS возвратов в продаваемый сток, кроме склада «Брак»). Живой месяц оценивается по уже сформированным недельным отчётам; ВБ дописывает отчёты в начале следующего месяца — страница их подхватывает. Данные Ozon и Яндекс — на соседних вкладках.
