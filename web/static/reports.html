Ozon — сводный отчёт по месяцам

Как читать этот отчёт

Данные 1:1 из раздела Финансы → Баланс личного кабинета Ozon (сверено с ЛК до рубля) + операционные показатели (заказы, возвраты, средний чек) и себестоимость из МойСклад. Каждое юрлицо — своя таблица, не суммируем. Строки зеркалят Баланс; столбцы — месяцы; справа доля от оборота и тренд. Подсветка — три блока относительно среднего сверенных месяцев: зелёное = выше среднего (хорошо), янтарное = ниже среднего (обратить внимание), без заливки — норма. Для расходов инверсия (ниже — лучше). Для ключевых статей — % оборота в ячейке. Закрытые месяцы — статика (со * — заморожены из оценки, ещё не сверены с Отчётом о реализации); два правых столбца — текущий месяц (оценка) и прогноз на конец месяца, живьём из БД.
